# Vocitap 语音输入工具 - 说明文档

欢迎使用 **Vocitap**，这是一个基于阿里巴巴 SenseVoiceSmall 模型的智能语音输入法。它支持按住左 Control 键说话，并在松开按键后自动将语音转换为文字“粘贴”到当前输入框。

## 📦 安装与存放位置说明

为了保持用户目录的整洁，Vocitap 采用了**“单 EXE + 云端补全”**的设计方案。运行后，您的系统会发生以下变化：

### 1. 核心程序
*   **Vocitap.exe**: 整个软件的唯一入口。内部已打包了所有的 Python 源代码，双击即可运行。

### 2. 数据存放路径 (AppData)
软件在运行时，会自动在您的 Windows 应用程序数据目录下创建存放配置和模型的空间：
*   **路径**: `%APPDATA%\Vocitap` (通常位于 `C:\Users\用户名\AppData\Roaming\Vocitap`)
*   **内容**:
    *   `config.json`: 存储您的个性化设置（如自启动开关、语气词过滤等）。
    *   `env_ready.tag`: 软件环境检测成功的标记文件。
    *   `models/`: **最重要的部分**。存放约 900MB 的 AI 模型文件。由于模型较大，首次运行如果没有此文件夹，软件会自动引导下载。

### 3. 系统环境依赖
Vocitap 启动器会检测您的 Python 环境。如果缺失必要的运行库（如 Torch, FunASR, PySide6 等），它会通过 `pip` 自动为您配齐，安装到您系统的 Python `site-packages` 目录下。

## 🚀 使用指南
1.  **启动**: 双击 `Vocitap.exe`。
2.  **首次初始化**: 如果是第一次运行，软件会显示“智能启动器”界面，自动检测并安装缺失的库和模型。
3.  **输入方式**: 
    *   按住键盘左侧的 **[Ctrl]** 键。
    *   对着麦克风说话。
    *   **松开 [Ctrl] 键**，文字会自动出现在光标处。
4.  **设置与退出**: 
    *   右键点击系统右下角托盘图标，选择“显示界面”。
    *   您可以勾选“开机自启动”或“自动剔除语气词”。
    *   点击“重置环境”可以彻底清理本地缓存（模型除外，需手动删除文件夹）。
